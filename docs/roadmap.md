# CCP Roadmap

## Short term (v0.1)
- Freeze JSON schemas (required vs optional)
- Finalize HTTP mapping
- Publish minimal message examples

## Mid term
- Reference implementation `ccp-python` (gateway/simulator)
- Reference implementation `ccp-esp32` (thin device)

## Long term
- Extensions for streaming, sensors, motion, and display
- Compatibility test suites
